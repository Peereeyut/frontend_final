NU Engineering Project Database System

This thesis focuses on designing and developing a database system for engineering projects with complex requirements at Naresuan University. Managing data for engineering projects with their increasing complexity and volume has become a challenge and necessity. A well-structured database system is crucial to efficiently collect, store, and retrieve project-related data, facilitating insightful analysis and informed decision-making throughout the project lifecycle.

The aim of this research is to design and implement a database system tailored for intricate engineering projects within the university. The system considers user requirements and employs appropriate database technologies to enhance workflow efficiency and decision-making. The user-friendly interface and robust security features of the database system cater to user needs effectively.

Research outcomes demonstrate the stability and efficiency of the developed database system. It showcases time savings in data retrieval and decision-making processes. User satisfaction with the system's usability further validates its success.

This study underscores the significance of developing a database system for complex engineering projects and presents research findings as guidelines for future database system development in engineering projects.


Keywords: Database system, Engineering projects, Complex requirements, Data management, Decision-making.

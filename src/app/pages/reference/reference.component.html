NU Engineering Project Database System

The reference website used in the NU Engineering Project Database System project (Engineering Project Database System, Naresuan University) is cited from the website https://www.ieee.org/. Angular 17 is used to develop the web application on the client side, and communication with the server is done using NodeJS, with the assistance of the Express.js Framework to manage APIs or data exchange between the client and the server, as well as the database. MySQL is utilized as the database to enhance the efficiency of the data storage process.

IEEE.org: For this project, ieee.org was utilized for its extensive database of engineering project templates and case studies, specifically to serve as a model for developing our own engineering database system for Naresuan University. Accessing ieee.org provided valuable insights into the structure and documentation of engineering projects, offering a reliable foundation for our database's development. It enabled the project team to incorporate best practices and standards in the creation of a robust and comprehensive engineering project database system, enhancing the educational and research capabilities at the university.

Angular 17: Angular is used in this project for client-side development. It provides a framework for building dynamic and interactive web applications. With Angular, the project benefits from features like data binding, modular architecture, and dependency injection, enhancing the overall user experience and development efficiency.

Angular Material: In this project, Angular Material is utilized to design a visually appealing and responsive user interface. It enables the use of well-designed components that are consistent with Material Design, making application development both efficient and straightforward. Angular Material aids in enhancing the application's usability and interactivity, facilitating a smoother development process and providing a richer user experience.

CSS: CSS plays a critical role in this project by providing the styling and layout for the web application. It is used to enhance the visual appearance of the application, including colors, fonts, and spacing, thus improving user interface design and user experience. Through CSS, the project achieves a polished and consistent look across different pages and components, ensuring the application is both attractive and easy to navigate for users.

Node.js: In this project, Node.js is utilized as the runtime environment for the server-side component. It allows the team to build scalable and efficient network applications, capable of handling numerous simultaneous connections with high throughput. Leveraging Node.js, the project benefits from its non-blocking, event-driven architecture, facilitating the development of fast, scalable APIs that interact with the client-side Angular application and MySQL database for data retrieval, storage, and manipulation.

MySQL: In this project, MySQL serves as the backbone of the engineering project database system at Naresuan University. It is utilized to store and manage project data efficiently, providing a reliable and scalable solution for handling large volumes of information related to engineering projects. By leveraging MySQL, the project team can implement a structured and organized database schema, ensuring data integrity and facilitating seamless data retrieval and manipulation. Additionally, MySQL serves as a valuable learning tool, allowing students and faculty to interact with real-world databases and gain practical experience in database management and query optimization.

References: Ieee.org, Angular 17, Angular Material, CSS, Node.js, MySQL

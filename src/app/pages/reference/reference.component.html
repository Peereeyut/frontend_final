NU Engineering Project Database System

The reference website used in the NU Engineering Project Database System project (Engineering Project Database System, Naresuan University) is cited from the website https://www.ieee.org/. Angular 17 is used to develop the web application on the client side, and communication with the server is done using NodeJS, with the assistance of the Express.js Framework to manage APIs or data exchange between the client and the server, as well as the database. MySQL is utilized as the database to enhance the efficiency of the data storage process.


References: Ieee.org, Angular 17, Angular Material, CSS, Node.js, Express, JSON Web Tokens, Bcrypt, MySQL
